# Task 1: Basic Mathematical Operations using Python

This project is a simple Python program written as part of an assignment.
It performs basic mathematical operations on two numbers entered by the user.

## Operations Performed
- Addition
- Subtraction
- Multiplication
- Division

## Input
The user is asked to enter two numbers using the keyboard.

## Output
The program displays the result of addition, subtraction, multiplication, and division
of the entered numbers.

## How to Run the Program
1. Open the Python file in PyCharm.
2. Run the program.
3. Enter two numbers when prompted.
