# Task 2: Personalized Greeting using Python

This project is a simple Python program created as part of an assignment.
It takes the user's first name and last name and prints a personalized greeting message.

## Task Description
The program performs the following steps:
- Takes the user's first name as input
- Takes the user's last name as input
- Combines both names into a full name
- Displays a greeting message using the full name

## Input
The user is prompted to enter:
- First name
- Last name

## Output
The program displays a personalized greeting message including the user's full name.

Example:
Hello, John Doe! Welcome to the Python program.

## How to Run the Program
1. Open the Python file in PyCharm.
2. Run the program.
3. Enter your first name and last name when prompted.
